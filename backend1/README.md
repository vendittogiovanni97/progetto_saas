# Progetto SaaS - Backend

Minimal Node.js Express Backend with TypeScript.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Setup**:
    - Copy `_env.local` to `.env`.
    - Adjust variables if necessary.

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    npm start
    ```

## API Endpoints

- `GET /health`: Check server status.
- `GET /`: Hello world message.
